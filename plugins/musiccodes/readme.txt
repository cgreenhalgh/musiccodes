== Description ==
Plugin for wordpress which gives a web-based graphical interface for authoring a
nd testing musiccodes, i.e. triggering actions from patterns within music.

Note: requires a supporting node.js-based service to perform low-level feature extraction. See https://github.com/cgreenhalgh/musiccodes

== Installation ==
1. Upload the directory `musiccodes` to the `/wp-content/plugins/` directory
1. Activate the plugin through the \'Plugins\' menu in WordPress
